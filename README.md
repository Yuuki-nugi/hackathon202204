## 機能  
・メッセージを送るとYahoo Shoppingから母の日に関連した商品を紹介する  
・紹介された商品が気に入ったかをyes/noで評価できる  
・全ユーザーの評価を集計し、ランキング上位３つの商品を紹介する  
・「下限:min,上限:max」の形式でメッセージを送ることで、商品価格の下限と上限を設定して商品を紹介する
